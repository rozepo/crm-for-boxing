# CRM секции бокса «Угол»

Staff-only MVP для учёта клиентов, абонементов и посещаемости.

Доступные интерфейсы:

- `/` — журнал тренера;
- `/admin` — клиенты и абонементы;
- `/admin/schedule` — календарь, разовые и повторяющиеся тренировки;
- `/book/login` — мобильная запись клиента по телефону.

## Запуск

```bash
npm install
npm run db:setup
npm run db:seed
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000). Демо-PIN: `1234`.

## Полезные команды

- `npm run test:core` — проверка списания и бесплатных посещений.
- `npm run db:seed` — вернуть демонстрационные данные.
- `npm run lint` — статическая проверка.
- `npm run build` — production-сборка.

PIN задаётся переменной `STAFF_PIN` в `.env`. Локальная SQLite хранится в `dev.db`.

## Публичный запуск

Перед публикацией нужно заменить SQLite на PostgreSQL, задать безопасный `STAFF_PIN` и подключить SMS-коды для клиентского входа. Текущий вход только по номеру телефона предназначен для закрытого пилота.
